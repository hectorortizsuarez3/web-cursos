
Plataforma Web de Cursos

Este proyecto es una plataforma de cursos online desarrollada como parte del proyecto intermodular del ciclo formativo de Desarrollo de Aplicaciones Web (DAW).
El sitio es de carácter educativo y demostrativo, sin finalidad comercial.

📌 Descripción del proyecto

La plataforma muestra una serie de cursos disponibles (HTML & CSS, Bases de Datos y JavaScript), junto con su descripción, duración, precio y fechas de inicio/fin.
Incluye además:

Preguntas frecuentes (FAQ) con respuestas que se muestran u ocultan dinámicamente mediante JavaScript.

Formulario de contacto con validación en cliente (JavaScript) y en servidor (PHP).

Los formularios se almacenan en una base de datos MySQL gestionada con phpMyAdmin (no configurado envío por correo en esta versión).

Aviso legal que indica el carácter educativo del sitio.

⚙️ Tecnologías utilizadas

Frontend:

HTML5

CSS3

JavaScript (para interacción FAQ y validación de formularios en cliente)

Backend:

PHP (procesamiento de formularios y validación en servidor)

Base de datos:

MySQL (gestionada con phpMyAdmin)

Servidor local:

XAMPP (Apache + MySQL + PHP)
